# Variable Size Box

```python
variable_size_box()
"""
****
****
****
****
"""

variable_size_box(3, '@')
"""
@@@
@@@
@@@
"""

variable_size_box(5, '.')
"""
.....
.....
.....
.....
.....
"""
```
